# Nowaste Roadmap

## Phase 1 — Foundation & Validation (Month 1–2)

- MVP web client + API scaffold.
- RSVP link generation and guest capture flow.
- Rule-based prediction (shared between API and AI stub).
- Shared Investo PostgreSQL schema with `nw_*` tables.
- Invite delivery hooks for email / WhatsApp integrations.

## Phase 2 — Prediction Engine (Month 3–4)

- Collect anonymized consumption telemetry (100–500 events).
- Feature engineering: event type, menu, demographics, seasonality.
- Train ML model (initially in notebooks) and deploy behind `ai-service`.
- Experiment with TensorFlow.js for on-device recalibration.

## Phase 3 — Monetization (Month 5–6)

- Subscription tier (`₹99` / month) and one-off prediction purchase (`₹49`).
- Razorpay / Stripe integration on the client.
- Catering vendor marketplace with referral tracking.
- CSR toggle to donate surplus via Robin Hood Army / Feeding India.

## Phase 4 — Impact & Automation (Month 7–9)

- Google Calendar sync for automatic reminders.
- Smart vendor recommendations by cuisine and capacity.
- Donation logistics scheduling with NGO partners.
- Impact analytics: food saved, carbon offset, beneficiaries served.

## Phase 5 — Scale & Partnerships (Month 10–12)

- Partnerships with event halls, planners, and catering SaaS.
- Public impact dashboards and marketing campaigns (“Nowaste Challenge”).
- National rollout strategy with localization backlog.
